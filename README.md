# P04_Persona_Chatbot

This project aims to develop a persona-based chatbot by modifying component within the Transformer architecture. Following the work of Chen et al., 2023: https://doi.org/10.1609/aaai.v37i11.26489 their paper proposed using memorize entialment and discourse relations to produce a persona consistent dialgues, this project aims to replicate their work in a different lightweight transformer model. Here Blenderbot was used. 

Dataset: CONVAI2 dataset
